# 🛒 Big Basket EDA Analysis

This mini project focuses on performing Exploratory Data Analysis (EDA) on the Big Basket product dataset to understand product pricing, discounts, ratings, and category distributions. The project applies Python-based analytical techniques to transform raw e-commerce data into meaningful business insights.

---

## 📌 Project Objective

The objective of this project is to explore the Big Basket dataset and analyze product characteristics, pricing strategies, discounts, and customer ratings through statistical analysis and data visualization.

---

## 📚 Dataset Information

* **Source:** Dataset provided as part of my Data Analytics training program.
* **Domain:** E-commerce / Retail Analytics
* **Dataset:** Big Basket Product Dataset
* **Industry Focus:** Online Grocery Retail
